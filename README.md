# Dismissible Widget App

A Flutter app demonstrating dismissible widgets with confirmation dialogs and undo functionality.

## Features
-  Swipe right to archive items (teal background)
-  Swipe left to delete items (red background)
-  Confirmation dialogs for both actions
-  Undo functionality via SnackBar
-  Material 3 dark theme

## How to Run
```bash
flutter pub get
flutter run
```

## Demo
